
** What is Python ? **

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

Python ek high-level, interpreted, object-oriented programming language hai jo Guido van Rossum ne 1991 mein develop ki thi. Yeh simple syntax, readability, aur versatility ki wajah se duniya ki sabse popular programming languages mein se ek hai.



** Used **
1) web development (server-side),
2) software development,
3) mathematics,
4) system scripting.
 
 ** Features **
✅ Simple & Readable Syntax → Python ka code human-friendly hota hai
✅ Interpreted Language → Direct execution without compilation
✅ Object-Oriented & Functional → OOP aur functional programming dono support karta hai
✅ Dynamically Typed → Variable ka type likhne ki zaroorat nahi hoti
✅ Extensive Libraries → Data science, AI, web dev, automation, har field ke liye modules available hain
✅ Cross-Platform → Windows, Linux, macOS, Android sab pe chal sakta hai
✅ Automation Friendly → DevOps, scripting, aur task automation ke liye best hai
✅ Large Community Support → Har problem ka solution online available hota hai
